XML Rules

Rule:1 XML Documents must have a Root Element. XML documents must have at lest one root element that is a parent of all elements: i.e;

Rule:2 XML prolog is optional (not mandatory) but if used it must come first in a document. Avoid errors you need to specify the encoding used, or save the file as a UTF-8(UTF-8 is the default character for (XML, HTML5, CSS, JavaScript, PHP, and SQL). XML documents can contain international characters, like Norwegian øæå or French êèé.

Rule:3 XML Elements must have closing tag. In XML it is "illegal" to not use the closing tag, all the elements must have a closing tag i.e;

Rule:4 XML tags are case sensitive. The tag "Letter" is diffrent from "letter", opening and closing tags must be written with the same case i.e;

Rule:5 XML elements properly nested. means that since the i element is opened inside the b element, it must be closed inside the b element.
